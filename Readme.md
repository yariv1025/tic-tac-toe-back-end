# Tic-Tac-Toc Game

> A Node.js project

Restful CRUD API for a simple Tic-Tac-Toe game application using Node.js
, Express and MongoDB.

## How to run the back-end server

Install dependencies
```bash
npm install
```

Run Server
```bash
node server.js
```

You can browse the apis at <http://localhost:3000>
